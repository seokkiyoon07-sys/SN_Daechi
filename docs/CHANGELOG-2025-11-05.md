## ✨ 주요 기능 구현

### 1. AI 시스템 탭
- **SNarGPT 시연 인터페이스**
  - 맥북 브라우저 UI 디자인으로 구현
  - 4가지 시연 모드: 문제 질문, 문제 검색, 정보 검색, 학원생활
  - 실제 수학 문제 풀이 예시 (2026학년도 9월 평가원 미적분 30번)
  - KaTeX를 이용한 수학 수식 렌더링
  - 두 가지 풀이 방법 제공 (대수적 풀이, 그래프 풀이)

- **AI 시스템 소개**
  - SNarGPT: 수능 전문 생성형 AI (정답률 99.9%)
  - SNarVIS: 학습 일정 및 오답 관리 AI 비서
  - SNarGEN: 평가원급 문제 생성 AI (출시 예정)

### 2. 데이터 시스템 탭
- **SNarlink (방화벽 + 학습량 측정 AI)**
  - 이미지 슬라이더 (3개 샘플)
  - 좌우 네비게이션 버튼
  - 인디케이터 표시

- **성적/오답데이터 시스템**
  - SNarOCR 이미지 클릭 확대 기능
  - 상세 분석 모달:
    - 전체 기본 통계 (250명, 평균 88.5점)
    - 문항별 오답률 분석 (1-30번)
    - 오답률 색상 코딩 (녹색/노란색/주황색/빨간색)
    - 상위 5개 난이도 문제
    - 과목별 통계 (국어, 수학, 영어, 과학탐구)
    - 수학 상세 통계 및 점수 분포 차트
  - 개인 성적표 모달:
    - SN고요의숲 브랜딩
    - 학생 정보 (이름, 학번, 회차)
    - 과목별 원점수, 표준점수, 백분위, 등급
    - 과목별 오답 번호 표시
    - 고성취 학생 데이터 (국어 96, 수학 96, 영어 94, 과학 48/47)

- **문제 데이터 & 입시 데이터**
  - 각 시스템 설명 및 기능 소개

  - 대치역 도보 3분 거리

### 4. FAQ 섹션 (AI 시스템 탭 하단)
- **SNarGPT vs ChatGPT 비교**
  - 핵심 답변: "ChatGPT는 세상을 아는 AI, SNarGPT는 학생을 아는 AI"
  - 비교표: 목적, 학습 데이터, 언어 이해, 답변 스타일
  - 기술 아키텍처 비교:
    - ChatGPT: General LLM (넓게 얕은 형태)
    - SNarGPT: Vertical AI (수능 특화, 문항 파싱, 선지 논리 트리, 오답패턴 연동)
  - 데이터 파이프라인 차이:
    - ChatGPT: 2024년까지 인터넷 데이터 (정적)
    - SNarGPT: SN고요의숲 실시간 학습 데이터 (동적)

## 🐛 해결된 이슈

### 1. 개발 서버 오류
- **문제**: React Client Manifest 모듈을 찾을 수 없는 오류
- **원인**: `.next` 빌드 캐시 손상
- **해결**:
  ```bash
  rm -rf .next
  rm -rf node_modules/.cache
  npm run dev
  ```

### 2. 포트 충돌
- **문제**: 포트 3000이 이미 사용 중
- **해결**: 기존 프로세스 종료 후 재시작
  ```bash
  taskkill //F //PID [PID번호]
  npm run dev
  ```

### 3. Vercel 배포 캐시 문제
- **문제**: 배포는 성공했으나 오래된 버전 표시
- **해결**: 강제 재배포로 캐시 클리어
  ```bash
  npx vercel --prod --force
  ```

### 4. ESLint 빌드 오류
- **문제**: 따옴표 escape 관련 16개 오류
- **해결**: 모든 텍스트 내 따옴표를 HTML entity로 변환
